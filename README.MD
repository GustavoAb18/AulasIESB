* Como consigo usar o código fornecido?

## Se você quer ver sobre os códigos de JavaScript:
1. Instale Node.js nesse site: [Node.js](https://nodejs.org/pt-br/download), e clique no **Instalador Windows (.msi)**;
2. Depois de instalado, navege o código que você baixou;
3. Por fim, execute com o comando `node 'name.js'`, e ele irá rodar.
